# spring-boot 오픈마켓 프로젝트

## 메인 기능
1. 회원가입
- 유효성검증
- 아이디 not null, PK
- 패스워드 not null, 특수문자 영대소문자, 숫자 조합 8자 이상
- 이름 not null
- 닉네임 not null
- 핸드폰번호

2. 로그인
- 아이디, 패스워드를 통한 로그인
- 둘 중 하나가 틀리면 로그인 정보 확인 alert 알림

3. 상품 업로드
- 이미지 업로드
- 이미지 업로드는 여러 이미지를 업로드할 수 있지만, 대표 이미지는 첫번째 이미지로 설정
- 상품 설명
- 가격 설정

4. 상품 문의
- 실시간 채팅

5. 상품 구매
- 상품 구매시 구매된 상품은 품절 표시

6. 상품 판매
- 판매자는 보유 예치금으로 판매 가격 입금받기
